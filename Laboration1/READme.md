# Movie Recommendation Lab

## Overview
This lab implements a simple movie recommendation system using the MovieLens dataset.  
The goal is to explore the data, prepare a dataset, and build a basic recommendation approach.

The following files are used:

- `movies.csv`
- `ratings.csv`
- `tags.csv`
